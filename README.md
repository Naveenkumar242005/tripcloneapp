# Travelora – Tripadvisor Clone

Travelora is a travel and tourism website inspired by Tripadvisor, built using HTML5 and CSS3.

The project features a modern travel-focused user interface with destination discovery, experience recommendations, category-based browsing, image sliders, and a responsive layout design.

## Features

- Modern and clean travel website UI
- Destination and experience recommendation sections
- Image slider/banner section
- Category-based travel exploration
- Search bar interface
- Professional footer section
- Built with Flexbox layouts
- Mobile responsiveness improvements planned

## Technologies Used

- HTML5
- CSS3
- Font Awesome
- Google Fonts (Poppins)

## Future Updates

- Responsive design for all screen sizes
- JavaScript-powered interactive slider
- Advanced search functionality
- Dark mode support
- Dynamic travel destination data
- User reviews and ratings section
- Backend integration for booking features
